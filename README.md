## Install

* zipでダウンロードして展開するかgitでclone
* Chromeの「拡張機能を管理」を開く
* デベロッパーモードをオン
* 「パッケージ化されていない拡張機能を読み込む」を押し、buildフォルダーを選択

---

This project was bootstrapped with [Chrome Extension CLI](https://github.com/dutiyesh/chrome-extension-cli)
